# Data Role Hiring Trends & Skill Demand Analysis

## Project Overview

This project analyzes 987 LinkedIn job postings across data-related roles to identify hiring trends, skill demand patterns, and role-based skill segmentation.

The objective was to understand how technical requirements differ across roles such as Data Analyst, Data Engineer, Data Scientist, ML Engineer, and Business Analyst.

---

## Problem Statement

What skills are most in-demand in the data job market, and how do these requirements vary across different data roles?

---

## Tools & Technologies Used

- Python (Pandas)
- Power BI
- DAX
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis

---

## Key Insights

- SQL is required in over 71% of job postings, making it the most foundational skill.
- Python dominates Data Scientist (88%) and ML Engineer (83%) roles.
- Data Engineers show the highest AWS demand (52%).
- Data Analyst roles rely heavily on BI tools like Tableau (49%) and Power BI (40%).
- Machine Learning is concentrated primarily in ML Engineer and Data Scientist roles.
